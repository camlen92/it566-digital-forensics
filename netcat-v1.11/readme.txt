 Name
  Netcat v1.11
 Description
  Netcat is used to create TCP channels. It is typically used by network admins, pen-testers, and programmers for network debugging.
 Your personal review
  Netcat is a rather old software but it robust in functionality and is highly dependable. I find it very useful to transfer data and extract data over a network quickly.
 Your personal notes on usage
  You can create a TCP listener simply by running the following command: nc -v -l -p <port number>
  The information heard by the listener can be piped using the following addition to the code above: > file.txt
