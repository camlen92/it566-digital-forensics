? Name
  Fport
? Description
  Fport is a network analysis tool that allows the user to view all processes using an open port.
? Your personal review
  Fport has been useful in identifying what processes could have opened a port and what port they are listening on. 
? Your personal notes on usage
  Simply type fport to get a list of all processes and what ports they are listening on.
